# homebrew-tab

A collection of formulas for my projects

## Update the SHA

If updating Flamingo, follow the below steps

```shell
curl -L -o flamingo-x.x.x.tar.gz https://github.com/spaghettifunk/flamingo/archive/refs/tags/vx.x.x.tar.gz
shasum -a 256 flamingo-x.x.x.tar.gz
```

Update the SHA value in the formula.
